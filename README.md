# Brain Tumor Detection AI
Hausarbeit für das Modul KI Grundlagen und Plattformen.

## Beschreibung
Dieses Programm nutzt ein Convolutional Neural Network (CNN), um Hirntumore in MRT-Bildern zu erkennen.

## Installation
- Installiere die benötigten Pakete: `pip install -r requirements.txt`
- Zum Testen eines Bildes führe `predict.py` aus.

## Datenquelle
Die Bilder stammen vom Kaggle Datensatz: https://www.kaggle.com/datasets/deeppythonist/brain-tumor-mri-dataset?resource=download (Mohammad rasol esfandiari)
